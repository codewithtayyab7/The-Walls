# The Walls

**The Walls** is a fan-made **Attack on Titan** project dedicated to bringing the world of the series to life through a modern and interactive web experience.

As a long-time admirer of *Attack on Titan*, I started this project to combine my passion for the franchise with web development. The goal is to create an immersive platform that explores the characters, Titans, history, and world of the series while providing engaging features for the community.

This project is actively under development. New features, improvements, and code updates will be pushed regularly, so feel free to check back often and follow the progress.

---

## Development Log

### Day 1 — 20/06/2026

Today marks the official beginning of **The Walls** project.

#### Progress Completed

* Created the initial landing/demo page using **HTML, CSS, and JavaScript**.
* Developed a single-page application containing approximately **907 lines of code**.
* Designed and implemented the core structure of the website.
* Established the foundation for future content, styling, and interactive features.

#### Current Sections Implemented

1. **Characters Database**

   * Information and details about major and supporting Attack on Titan characters.

2. **Titan Encyclopedia**

   * Complete collection of Titans along with their abilities, history, and descriptions.

3. **Attack on Titan Timeline**

   * A chronological overview of major events throughout the series.

4. **World Map**

   * Interactive representation of the Attack on Titan world and important locations.

5. **Personality Quiz**

   * A quiz section designed to determine which Attack on Titan character best matches the user.

6. **Community Hub**

   * A dedicated space for fans to interact, share ideas, and discuss the series.

---

### Day 2 — 21/06/2026

Today focused on modernizing and scaling the project's architecture.

#### Progress Completed

* Migrated the entire application from a standalone **HTML, CSS, and JavaScript** implementation to a **React-based MERN Stack architecture**.
* Initialized the frontend using **Vite** for fast development and optimized build performance.
* Integrated **Tailwind CSS** for a scalable and maintainable styling system.
* Reorganized the project into reusable React components.
* Prepared the architecture for future backend integration and dynamic data handling.

#### Notes

* No major visual changes were introduced in this phase.
* The main focus was improving scalability and maintainability while preserving the existing UI and functionality.

#### Tech Stack (Current)

**Frontend**

* React (Vite)
* Tailwind CSS

**Planned Backend**

* Node.js
* Express.js
* MongoDB

### Day 3 — 22/06/2026

Today was focused on improving the **UI/UX structure and navigation system** of the application.

#### Progress Completed

* Improved UI interactions, including hover effects and cursor behavior for better user experience.
* Refined the overall visual responsiveness and interactivity of components.
* Restructured the application architecture by separating content into dedicated pages instead of a single-page layout.

#### New Page Structure

The application is now organized into multiple pages:

* **Home**
* **Characters**
* **Titans**
* **Timeline**
* **Rumbling Quiz**
* **Community**

Each section now has its own dedicated route/page, improving scalability, navigation clarity, and maintainability.

#### Notes

* The focus of this update was structural and UX improvement rather than adding new features.
* Data separation from the homepage is an ongoing process to make the project more modular and production-ready.

#### Next Goals

* Implement backend APIs for dynamic data loading.
* Connect MongoDB for storing characters, Titans, and timeline data.
* Enhance routing and page transitions.
* Improve UI polish and animations.
* Start implementing community interaction features.
